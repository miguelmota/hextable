# hextable

> Bash script to render a simple hex table

## Install

```bash
wget https://raw.githubusercontent.com/miguelmota/hextable/master/hextable.sh
chmod +x hextable.sh
sudo mv hextable.sh /usr/local/bin/hextable
```
